# ❤一个人名为Beatz的音乐网站 ❤

技术使用: [React](https://reactjs.org/), [dva.js](https://dvajs.com/), [umi.js](https://umijs.org/docs)

UI框架:  [material-ui](https://material-ui.com/)

enjoy😊
 
## 环境准备

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```



### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```
